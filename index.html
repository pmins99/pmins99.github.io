TinaCMS — Path Traversal via relativePath (CWE-22)

TinaCMS v3.3.1의 GraphQL resolver에서 사용자 입력값인 relativePath가 별도 검증 없이 path.join()에 전달되는 구조. ../ 시퀀스를 통해 허용되지 않은 서버 파일에 접근 가능한 Path Traversal 취약점을 발굴, 패치까지 직접 제안하여 공식 Security Advisory에 크레딧 등재.

Target TinaCMS v3.3.1
Type CWE-22 Path Traversal
Platform GitHub Security Advisory
View Advisory on GitHub →
pathtraversal.poc.test.ts
// Vulnerable code
const realPath = path.join(collection.path, relativePath);
// Exploit
relativePath = "../../SAFE_OUTSIDE/poc.mdx"
// Patch
if (back.startsWith('..') || path.posix.isAbsolute(back)) {
  throw new Error('Invalid relativePath');
}

취약점 연구

오픈소스 CMS 6종을 대상으로 직접 소스코드 분석을 통해 발굴한 취약점 목록입니다. 모든 취약점은 책임있는 공개(Responsible Disclosure) 절차를 거쳤습니다.

Directus v11.14.0
High
CWE-284 · Improper Access Control

비관리자에게 filename_disk 쓰기 권한이 있을 경우, 자신의 파일 포인터를 피해자 스토리지 키로 변경 후 삭제하여 피해자 파일을 원격에서 삭제 가능. 데이터 손상 및 레코드-파일 불일치 유발.

PR 리뷰 중 · 다음 릴리즈 반영 예정
Paperless-ngx v2.20.2
High
CWE-79 · Stored XSS

safeHtml 파이프에서 bypassSecurityTrustHtml을 사용해 Angular 기본 sanitization을 우회. [innerHTML]을 통한 저장형 XSS 발생. GitHub Security 제보 후 공식 패치 완료.

패치 완료 · Vendor Acknowledged
Chatwoot v3.0.0 ~ v4.8.0
High
CWE-79 · Stored DOM XSS

Help Center 아티클 헤딩 텍스트가 검증 없이 HTML 속성(title)에 삽입되고 innerHTML+=으로 재해석. 속성 탈출 및 이벤트 핸들러 주입 가능. GitHub Security 제보 후 패치 완료.

패치 완료 · Vendor Acknowledged
Chatwoot v4.8.0
High
CWE-346 · Origin Validation Error

위젯의 postMessage 통신에서 targetOrigin="*" 및 event.origin 검증 부재. 세션 토큰(cw_conversation) 주입을 통한 대화 세션 고정·탈취 가능. 피해자 메시지가 공격자 대화로 노출.

회신 수령 · 패치 미완료
ERPNext v15.91.0 ~ v15.93.1
High
CWE-79 / CWE-918 · XSS + SSRF (복수)

eval() 기반 XSS 4건(Number Card, Letter Head, Awesome Bar, Webpage Content), SSRF 2건(Newsletter, Code List) 발굴. 정규식 우회 기법 포함. 벤더에 제보 완료, 공식 회신 대기 중.

제보 완료 · 회신 대기
Mealie v3.6.1
Critical
Broken Access Control + JWT Forgery

일반 사용자가 파일 다운로드 기능으로 서버 DB 및 JWT 서명 키 탈취 가능. 탈취한 키로 관리자 Access Token 위조하여 인증 우회. 선제 제보자에 의해 패치 완료.

패치 완료 (선제 보고)
Discourse v2026.1.0
Medium
CWE-346 · Confused Deputy (Origin Validation)

AI Artifact 기능의 postMessage 처리에서 event.origin 및 event.source 검증 누락. 외부 출처 페이지가 피해자 권한으로 Artifact KV API 호출 가능. HackerOne 제보, 버그바운티 대상 외 회신.

Medium
MITM · SSL Certificate Verification Disabled

Google OAuth 연동 시 CURLOPT_SSL_VERIFYPEER가 false로 설정되어 가짜 인증서 수용. 네트워크 MITM으로 OAuth 통신 탈취·조작 가능. KISA 제보, 위험도 낮음 평가.

KISA 제보 완료
Strapi v5.31.3
Medium
CWE-1333 · ReDoS

Admin Webhooks URL 검증 정규식에 중첩 반복 구조 존재. 특정 입력에서 과도한 백트래킹으로 CPU 급증 및 Node.js 이벤트 루프 블로킹. URL 파서 기반 검증으로 패치 제안. 벤더 회신 대기 중.
